HTML TAGS, ATTRIBUTES AND ELEMENTS

This document introduces you to the basic html tags used for structuring contents of html web pages. It shows how to add texts, headings, lists, tables, hyperlinks and images to webpages. Go through the code used in producing this web page and try reproducing it to your own specifications. HTML tags and attributes are a lot more than those stated here, but for an introductory crash course, these are necessary.

  • Tags

    The basic structure of an HTML document includes tags, which surround content and apply meaning to it. The purpose of HTML is to apply meaning, not presentation, and this example has now defined some fundamental elements of a web page.

    Closing tags

    The </body> and </html> put a close to their respective elements (more on elements in a moment).

  • Attributes

    Tags can also have attributes, which are extra bits of information. Attributes appear inside the opening tag and their values sit inside quotation marks. They look something like <tag attribute="value">Margarine</tag>.

  • Elements

    Tags tend not to do much more than mark the beginning and end of an element. Elements are the bits that make up web pages. You would say, for example, that everything that is in between (and includes) the <body> and </body> tags is the body element. As another example, whereas “<title>” and “</title>” are tags, “<title>Ladies Geeking Out</title>” is a title element.

    Tags: 'img', 'src'
